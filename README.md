# Pima Indians Diabetes Classification & Model Comparison

A high-level diagnostic analysis to predict whether a patient has diabetes based on clinical measurements, using a comparative approach across multiple machine learning algorithms.

##  Project Overview
This project aims to identify the most effective classification model for diabetes diagnosis. It involves rigorous data exploration and a benchmark of 10 different classifiers to determine which provides the most reliable predictive power in a medical context.

## Tech Stack & Algorithms
* **Ensemble Learning:** AdaBoost, XGBoost, LightGBM, Gradient Boosting, Random Forest.
* **Classical ML:** Logistic Regression, KNN, SVM, Naive Bayes, Decision Tree.
* **Evaluation Metrics:** Accuracy Score, AUC-ROC, Confusion Matrix.
* **Libraries:** Scikit-learn, LightGBM, XGBoost, Pandas, Seaborn, Matplotlib.

## Key Technical Steps
- **Exploratory Data Analysis (EDA):** Visualized correlations between features like Glucose/BMI and the target outcome.
- **Performance Benchmarking:** Created a systematic comparison table for all 10 models based on Accuracy and AUC.
- **Visualization:** Generated bar plots to compare AUC scores, highlighting the superior performance of boosting algorithms.

## Results Summary
- **Top Performers:** AdaBoost and LightGBM led the benchmark with the best balance of Accuracy and AUC.
- **Clinical Insight:** Glucose levels and BMI were identified as the strongest predictors for diabetes onset in this dataset.
